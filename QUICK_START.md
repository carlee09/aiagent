## ğŸ› Troubleshooting

### Issue: Sentiment analysis shows "Unknown"
**Solution**: Data might not have enough text content. Try increasing `--max-items`.

### Issue: Keywords look strange
**Solution**: YAKE extracts n-grams, so some keywords might be 2-word phrases. This is normal.

### Issue: Interactive mode fails
**Solution**: Make sure your AI model API key is valid and has sufficient quota.

### Issue: Comparison fails
**Solution**: Ensure the previous report file exists and is a valid markdown file generated by this tool.
